## API

Using official [disease.sh](https://disease.sh/) API

- [Global](https://disease.sh/v3/covid-19/all)
- [Continents](https://disease.sh/v3/covid-19/continents)
- [Countries](https://disease.sh/v3/covid-19/countries)
